Variant3D (Variants on 3D Structure)

An auto-updating database collecting and annotating protein point mutations on 3D structures

e.g. P53_HUMAN or P04637 or ENSP00000269305.4 or ENSG00000141510.11

Variant3D is an automatically weekly updated database collecting and annotating all human-related point residue variants on 3D protein structures. Built on genomic to structure mapping tool G2S, Variant3D systematically collects all point mutations mapping from human UniProt/Ensemble proteins in PDB. Aligning back to genomic locations, these mutations on 3D structures are annotated by structural annotation via analyzing PDB, variant annotation as DBSNP, genetics annotation as COSMIC, disease annotation as Clinvar/TCGA. Variant3D provides a user-friendly API with REST-inspired design, and it can be used by various clients and other bioinformatics tools for bringing 3D structures into the genomic variant analysis.

Highlights:

  1. • Collecting all human point mutations on 3D structure in the whole PDB.
  2. • Structure annotation by 2nd Structure, protein surface location, and sequence/structrue domain
  3. • 3D protein variants annotated by mapping dbSNP/COSMIC/TCGA on human genome
  4. • Automatically update every week after PDB updates
  5. • RESTful API for users and clients
